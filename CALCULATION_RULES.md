# 근무시간 계산 룰 (선택적 근로시간제 · 월 단위 정산)

> Suprema 근무제도 (2026-07-01 시행) 기준. SuperEarlyBird 계산기의 도메인 스펙.
> 출처: 사내 제도개편 공지 + 시스템 설명회 Q&A + 근태관리 시행세칙 신구조문 대비표 + 회사 시뮬레이션 슬라이드(6월).

---

## 1. 핵심 산식

```
월 기준근로시간(h) = floor(역일수 / 7 * 40) - (평일 공휴일수 * 8)
월 최대근로시간(h) = floor(역일수 / 7 * 52)
```

- **소수점은 버림(floor)**. 기준·최대 둘 다 동일 적용.
- `역일수` = 그 달의 달력상 총 일수 (28/30/31).
- `평일 공휴일수` = 그 달 **월~금**에 떨어지는 공휴일 수. **주말(토·일) 공휴일은 차감하지 않음.**
- 공휴일 1일 = **8h 차감** (정수라 floor 순서 무관).
- **최대근로시간은 공휴일을 차감하지 않음** — 휴일근무로 채울 수 있는 capacity이기 때문. 기준만 내려가고 최대는 유지되는 비대칭 구조.
- 정산 단위: **매월 1일 ~ 말일**.

### 근로자의 날 (5/1)
- 관공서 공휴일이 아니라 천문연 API에는 **안 나옴**.
- 회사가 유급휴일로 처리하면, 5/1이 평일일 때 **평일 공휴일로 8h 차감**. (앱에서는 토글로 처리, 기본 ON)

## 2. 휴게시간 (자동 차감)

```
rest(W) = 0.5 * floor(W / 4)        # 단위: 시간. W = 그날 실근무시간(h)
```

| 실근무 W | 자동 휴게 |
|---|---|
| W < 4h | 0h (단, 4h 근무일 신청 시 면제 가능 — 아래) |
| 4h ≤ W < 8h | 0.5h (30분) |
| 8h ≤ W < 12h | 1.0h (60분) |
| 12h ≤ W < 16h | 1.5h (90분) |

- **실근무 4시간당 30분**이 자동 차감. 밥을 빨리 먹거나 굶어도 근로로 인정 안 됨(자동 차감 고정).
- **4시간 근무일에 한해** 업무종료 후 휴게 면제 신청 가능. 그 외 근무일은 면제 불가.
- 12h 근무 시 휴게 1.5h는 분할 사용 가능, 1.5h 초과분만 시스템 등록.
- 권장 점심 11:30~12:30이나 개인 근무시간에 따라 분할 사용 가능.

### ★ 8시간 절벽 (cliff)
실근무가 **8.0h를 넘는 순간** 두 번째 30분 휴게 블록이 붙는다.
- 7h59m 근무 → 휴게 30분 → 체류 8h29m
- 8h00m 근무 → 휴게 60분 → 체류 9h00m
- → 1분 더 일하려다 **31분 더 묶임**. 스케줄 최적화의 핵심 분기점.

## 3. 체류시간 · 퇴근시각

```
체류시간 = 근무 W + rest(W)
퇴근시각 = 출근시각 + 체류시간
```
- 기본 출근 가정: **07:30** (집-회사 근접 활용). 인정 시간대 07:30~23:00 안에서 자율.
- 인정 시간대(07:30~23:00) 밖 근무는 사전 HR 협의 필요.

## 4. Core-time

| 요일 | Core-time |
|---|---|
| 월~목 | 10:00 ~ 15:00 |
| 금 | 10:00 ~ 12:00 |

- Core-time을 위반해 출근하지 않으면 **결근 처리**.
- 예외: 셋째 주 금요일(미출근 선택일)은 결근 처리 안 됨.

---

## 5. 셋째 주 금요일 (미출근 선택일)

- **정의(사내 기준)**: "세 번째 월요일이 있는 주의 금요일" = `3번째 월요일 + 4일`.
  (※ 단순히 그 달의 3번째 금요일이 아님에 주의)
- 월 **1회** 미출근 선택 가능. 계획 설정 후 미출근해도 **결근 아님**.
- 그날이 **공휴일이면 조정 없음**.
  - 예) 2026-07: 셋째 금 = **7/24** (3번째 월요일 7/20이 든 주의 금). 제헌절 7/17과는 **별개** → 7월은 7/17(공휴일) + 7/24(선택) 둘 다 가능.
- 미출근을 선택해도 **월 기준근로시간은 그대로 채워야 함** (다른 주에 분산 조정 가능).

```
function thirdFriday(year, month):   # month 1-based
    first = date(year, month, 1)
    offsetToMonday = (1 - first.weekday() + 7) % 7   # weekday: Sun=0..Sat=6 기준이면 보정
    thirdMonday = 1 + offsetToMonday + 14
    return date(year, month, thirdMonday + 4)         # +4 = 금요일
```

## 6. 연차 / 휴가

| 단위 | 시간 |
|---|---|
| 연차 | 8h |
| 반차 | 4h |
| 반반차 | 2h |

- 10h 차감이 필요하면 연차 + 반반차 조합.
- 연차 사용 시 **계획시간이 아니라 당일 인정근무시간에서 차감** (1일 연차 = 8h 차감).
- 건강검진휴가(0.5일) 등은 휴가타입 + 반차/반반차 조합으로 처리.

## 7. 연장 / 휴일 / 외근

- **연장근무 = 월 실근무시간 − 월 기준근로시간 − 고정연장(월 20h)**.
- 고정연장 **월 20시간**까지는 (포괄) 별도 수당 대상 아님, 그 이상부터 별도.
- **휴일근무**: 별도 신청, 신청/계획 시간 내 인정.
- **월 최대근무가능시간 초과 등록 불가** (필요 시 사전 HR 협의).
- **외근·해외출장(직출/직퇴)**: 1일 8h 인정. 복귀 후 추가근무 시 8h 초과분 인정 가능. 간주근로 적용 시 1일 최대 8h 원칙(예외는 사전 HR).

## 8. 운영 / 시스템

- 근무계획 변경은 **근무 당일까지** 가능, 이후는 근태조정신청(근무일 +1주 내 원칙).
- 승인 라인: 부서장 승인 (부서장 본인은 총괄 승인).
- 기본세팅값은 HR 일괄등록 → 개인이 직접 못 바꾸고 **계획에서 조정**.
- 2026-07 기본세팅 placeholder: (월~목) 07:30~18:30, (금) 07:30~12:00 → **본인 플랜으로 직접 덮어써야 함**.

## 9. 스케줄 전략 (계산기 로직)

```
소정근무일 = 평일수 - 평일공휴일수  ( - 1  if 셋째금 미출근 선택 )
S = 월 기준근로시간,  N = 소정근무일,  start = 출근시각
```

### A. 균등배분
```
W = S / N                # 매일 동일 근무
out = start + W + rest(W)
```
- N이 충분해 `W < 8`이면 **이미 최적** (모든 날 휴게 30분, 일찍 퇴근). 손대지 말 것.

### B. 8시간 절벽 회피
```
SHORT = 7 + 59/60        # 7h59m, 절벽 바로 아래
if S <= SHORT * N:        # 균등이 이미 W<8 → B의 이점 없음
    use A
else:
    x = ceil((S - SHORT*N) / (Lmax - SHORT))     # 롱데이 수 (Lmax: 몰아치는 날 최대근무)
    longLen = (S - SHORT*(N - x)) / x
    # (N-x)일: 7h59m 근무, 휴게 30분, 일찍 퇴근
    # x일: longLen 근무, 몰아치기
```
- **평일이 적은 달(소정 적음 → 균등 W ≥ 8)** 에서 유리: 대부분 칼퇴 + 며칠만 길게 → 월 총 체류시간 절약.
- 평일 많은 달은 A가 정답.

## 12. 엣지 케이스 메모

- **2월(28일)**: floor(28/7*40)=160h, 항상 평일 20일 기준. base가 정확히 8.0h라 공휴일 수와 무관하게 균등 퇴근 16:30 고정.
- **절벽 불연속**: 체류시간 함수는 W=8.0에서 8h29m → 9h00m로 점프. W를 8.0 직전(7h59m)에 두면 휴게 30분 유지.
- **셋째 금이 공휴일**: 조정 없음. 미출근 선택일 별도 부여 여부는 사례별(HR 확인 필요).
- **소정근무일이 적은 달(5·8월)**: 균등 시 W>8 강제 → 절벽 회피 전략이 총 체류시간 유리.
